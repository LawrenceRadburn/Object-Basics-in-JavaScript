Objects in JavaScript

Objects are a way to store key value pairs in JavaScript. The key is represented by a name. Whereas the value is typically represented as a number or a string.

In essence, Objects help us store highly related data together.

Moreover, we store store functions within objects. More on that later.

Object Literals

Object literals are the foundation are object syntax. Here is an example of a object with a body.

const car = {

        };

Now let's add our key value pairs.

const car = {
type: "passenger vehicle"
};

In the example above, our key is type and our value is "passenger vechicle".

Now let's add some more comprehensive data.

const car = {
type: "passenger vehicle",
wheels: 4,
engine: true,
max_passengers: 8
};

Notice how between each subsequent key value pair, we comma is required with the exception of the final key value pair.

Functions in Objects

We can add functions inside of our objects. Here is an example.

const car = {
type: "passenger vehicle",
wheels: 4,
engine: true,
max_passengers: 8,
sell: function() {
    return("I am sold.");
    },
};

We call this type of function and anonymous function because it is not bound to an identifier and thus cannot call itself in isolation.

Using Function to Return Objects

When beginning with functions, it's common to use functions to return primitive data types such as strings, numbers, and booleans.

In addition, we can use functions to return objects. Here is an example:

function Car(make, model, year, vin) {
this.make = make;
this.model = model;
this.year = year;
this.vin = vin;
};

Now that we have created the created our function, we can call the function and pass in our parameters inside the function.

In this example, our parameters are: make, model, year, and vin

Here's how that will work:

const hondaCivic001 = new Car("Honda", "Civic", 1972, "2HGEH2342NH5001972");
const hondaCivic002 = new Car("Honda", "Civic", 1973, "2HGEH2342NH5001973");
const hondaCivic003 = new Car("Honda", "Civic", 1974, "2HGEH2342NH5001974")

If we would like to display our newly created object to the DOM (Document Object Model), we can create another function to do so.

function displayCar(car) {
output.innerText += `Make: ${car.make}, Model: ${car.model}, Year: ${car.year}, VIN: ${car.vin}\n`;
};

Now lets call our new function and pass in our varibale we initalized earlier.

const hondaCivic002 = new Car("Honda", "Civic", 1973, "2HGEH2342NH5001973");
